# Onboarding

Step-by-step guide for new team members.

## Week 1: Setup

- Access to all required systems and tools
- Introduction to team members and stakeholders
- Development environment setup
- Review of coding standards and conventions

## Week 2: Orientation

- Architecture walkthrough
- Codebase tour with a buddy
- First small task assignment
- Team process and ceremony overview

## Week 3: Integration

- First code review participation
- Sprint planning involvement
- Knowledge base contribution
- Feedback session with manager
